# Reagent Tracker - программа учета реактивов

## Описание
Консольное приложение для учета реактивов в лаборатории. Позволяет добавлять реактивы, партии, отслеживать движения и формировать отчеты.

## Команды
- `reag_add` - создать новый реактив
- `reag_list [--q ТЕКСТ]` - список реактивов (с поиском)
- `batch_add` - добавить новую партию
- `batch_list [ID]` - список партий (для реактива или все)
- `batch_show <ID>` - карточка партии
- `move_add <ID>` - добавить движение
- `move_list <ID> [--last N]` - история движений
- `batch_update <ID> поле=значение...` - обновить партию
- `batch_archive <ID>` - архивировать партию
- `stock_report [--expires-before ДАТА]` - отчет по складу
- `clear_data` - удалить все данные
- `save` - сохранить в файл
- `help` - справка
- `exit` - выход
